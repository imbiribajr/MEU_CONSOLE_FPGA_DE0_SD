Git Cheat Sheet - Projetos FPGA

Clique no botao ao lado de cada comando para copiar.

1) Configuracao Inicial (Identidade)

Executar apenas uma vez no computador novo.

git config --global user.name "Reinaldo Imbiriba"
git config --global user.email "j.imbiriba@gmail.com"
git config --global credential.helper manager
git config --list

2) Iniciando o Projeto (Local)

Executar na pasta raiz do projeto (onde esta o .gitignore).

git init
git add .

3) Conectando ao GitHub (Primeiro Envio)

Substitua a URL abaixo pelo link do repositorio que voce criou no site.

git commit -m "Primeiro commit: Projeto FPGA Tetris"
git branch -M main
git remote add origin https://github.com//NOME_DO_REPO.git
git push -u origin main

4) Fluxo de Trabalho (Dia a Dia)

Sempre que alterar e quiser enviar para o GitHub.

Passo Comando Descricao
Preparar git add . Prepara todos os arquivos modificados.
Gravar git commit -m "Mensagem" Salva no historico local.
Enviar git push Envia as alteracoes para o GitHub.

Comandos Uteis

Comando O que faz
git status Mostra quais arquivos foram mudados.
git log Mostra o historico de commits.
git pull Baixa atualizacoes da nuvem.
Dica: abra este arquivo no navegador para copiar os comandos com um clique.
